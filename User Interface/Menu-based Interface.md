# USER INTERFACE

### A. Choice of User Interface (UI)
For the Automated Digital Forensics Evidence Collection Tool, we use a hybrid user interface combining both Menu-Based Interface and Direct Manipulation Interface.

## 1.	Menu-Based Interface
A menu-based interface allows users to interact with the system by selecting options from a set of predefined menus.

**Why it is used in our system?**
-	**Structured Navigation**:   The system contains multiple modules such as:
  -	Case Management
  -	Evidence Upload
  -	Evidence Analysis
  -	Chain of Custody
  -	Report Generation
-	A menu system helps users navigate these modules easily.
-	**Role-Based Access** as users can choose aming the listed roles
-	The system has **3 user roles**:
  - System Administrator
  -	Forensic Investigator
  -	Forensic Analyst
-	Each role is provided with different menu options.
-	**Reduced Errors**:   Users choose from predefined actions rather than typing commands, which reduces input errors.
-	Menu-driven systems are intuitive and easy for investigators who may not have advanced technical knowledge.
