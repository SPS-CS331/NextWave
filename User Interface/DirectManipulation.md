## 2. Direct Manipulation Interface

A direct manipulation interface allows users to interact directly with visual objects on the screen using actions like clicking, dragging, and dropping.

<img width="1409" height="640" alt="image" src="https://github.com/user-attachments/assets/daf250aa-7846-4f53-b79b-8c75ea440e58" />


### Why it is used in our system?

- **Evidence Upload:**  
  Investigators can drag and drop evidence files directly into the system.

- **Interactive Data Visualization:**  
  Analysts can interact with tables and evidence records.

- **File Handling:**  
  Evidence files can be:
  - Opened  
  - Previewed  
  - Moved  
  - Selected directly from the interface  

- **Improved User Experience:**  
  Direct interaction with visual elements makes the system more intuitive and faster to use.

### Example Direct Manipulation Actions

- Drag and drop evidence files into the upload area  
- Click evidence items to view details  
- Select analysis options using buttons
